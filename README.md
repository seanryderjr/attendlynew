# Attendly Full

Railway-ready Attendly replacement with Express, PostgreSQL, and a full browser UI.

## Railway variables

Set these on your app service:

- `DATABASE_URL` from Railway PostgreSQL
- `JWT_SECRET` any long random text
- `ADMIN_EMAIL` default `admin@attendly.local`
- `ADMIN_PASSWORD` default `admin123!`
- `ADMIN_NAME` default `Admin User`

Start command: `npm start`
Root directory: blank if files are at repo root.

## Default login

admin@attendly.local / admin123!
